# Fonts Directory

## Required Font Files

Please add the following font files to their respective directories:

### Season Mix (src/assets/fonts/seasonmix/)
- SeasonMix-Bold.woff2
- SeasonMix-Bold.woff
- SeasonMix-ExtraBold.woff2
- SeasonMix-ExtraBold.woff

### Matter (src/assets/fonts/matter/)
- Matter-Regular.woff2
- Matter-Regular.woff
- Matter-Medium.woff2
- Matter-Medium.woff
- Matter-SemiBold.woff2
- Matter-SemiBold.woff
- Matter-Bold.woff2
- Matter-Bold.woff

## Font Usage
- **Season Mix**: Used for all headings (VajraOpz logo, page titles like "Text to Speech", "Billing", etc.)
- **Matter**: Used for all other text (nav items, labels, descriptions, buttons, body text)

## Fallback
DM Sans is used as a fallback font until the custom fonts are added.
